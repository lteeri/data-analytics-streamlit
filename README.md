# data-analytics-streamlit

## Main assignment 06 - Streamlit

Lapland hotel stats: A Streamlit app for visualizing Statistics Finland municipality-level hotel stats from

https://pxdata.stat.fi/PxWeb/pxweb/en/StatFin/StatFin__matk/statfin_matk_pxt_11lm.px/

(Monthly hotel capacity and nights spent by municipality by Municipality, Information and Month)


Create a new public GitHub repo. Deploy your Streamlit app on Streamlit Community Cloud. Deployment is part of grading.

Submit both links:

- GitHub repo URL
- Streamlit Cloud app URL

Also, remember to share your GitHub repo with datamikko (my work GitHub username).

Code requirements

- Load one municipality via saved-query CSV URL (PxWeb).
- Show a data table for that municipality.
- Simple overall trend (line chart) for a chosen metric.
- Yearly totals (bar chart) for nights spent.
- Domestic vs Foreign: datatable and small line chart.
- Compare with another municipality: load a second CSV + overlay on a chart.
- Download button to export the comparison data.

Add text in between the charts in the streamlit app. Communicate your findings in the text fields.

Screen recording video is not needed.
